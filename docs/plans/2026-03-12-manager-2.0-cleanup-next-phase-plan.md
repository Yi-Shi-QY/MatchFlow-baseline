> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Architecture:** Keep the current manager gateway and runtime-domain-pack architecture as the single source of truth. Remove the remaining old page-owned command-center state, rename leftover football-specific legacy workflow symbols, and tighten the hot-pluggable runtime-domain/data-source contract without reopening old compatibility layers.

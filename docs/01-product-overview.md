# Product Overview / 产品概览

## EN

## 1. What MatchFlow Is

MatchFlow is a local-first football analysis application built with:

1. React + Vite for web UI.
2. Capacitor wrappers for Android and iOS.
3. Multi-agent orchestration for structured analysis output.
4. Optional match-data backend (`match-data-server`) for real-time or managed data.

## 2. Core Value

1. Turn raw match information into structured, explainable analysis.
2. Generate sectioned analysis with optional visual animation blocks.
3. Keep offline-friendly user history and resume capability.
4. Support deterministic planning routes from selected data sources.

## 3. Main Capability Areas

1. Data input:
   - Built-in mock matches.
   - Server-fetched matches.
   - Custom imported/scanned payloads.
2. Analysis runtime:
   - Planner route resolution (template or autonomous).
   - Specialist agent segments.
   - Tag extraction and final summary.
3. Visualization:
   - Parameterized Remotion templates.
4. Extensibility:
   - Agent/skill/template manifests.
   - Hub-based auto-install for missing extensions.

## 4. Current Direction (2026)

1. Make data-source selection fully declarative.
2. Keep per-agent model routing configurable (global mode + config mode).
3. Strengthen extension validation/version lifecycle for future hub ecosystem.
4. Improve maintainability by modular architecture and clear doc ownership.

## ZH

## 1. MatchFlow 是什么

MatchFlow 是一个本地优先的足球比赛分析应用，技术组成包括：

1. React + Vite Web 前端。
2. Capacitor Android/iOS 封装。
3. 多 Agent 编排的结构化分析流程。
4. 可选的比赛数据服务端（`match-data-server`）用于真实数据与运维管理。

## 2. 核心价值

1. 将原始比赛信息转成结构化、可解释的分析结论。
2. 生成分段分析内容，并可附带动画可视化块。
3. 提供离线友好的历史记录与异常中断后续跑能力。
4. 基于数据源选择实现可预测、可控的规划路由。

## 3. 主要能力域

1. 数据输入：
   - 内置 Mock 赛事。
   - 服务端赛事拉取。
   - 自定义导入/扫码数据。
2. 分析运行时：
   - 规划路由决策（模板式 / 自主式）。
   - 专项 Agent 分段执行。
   - 标签提取与最终总结。
3. 可视化：
   - 参数化 Remotion 模板渲染。
4. 可扩展：
   - Agent/Skill/Template Manifest。
   - Hub 自动安装缺失扩展。

## 4. 当前方向（2026）

1. 把数据源选择界面和行为全面声明式化。
2. 稳定每个 Agent 的独立模型路由（全局模式 + 配置模式）。
3. 增强扩展校验和版本生命周期，为后续 Hub 生态做准备。
4. 通过模块化架构与统一文档提升维护效率。
